Class: StringDictionary

sntls. StringDictionary

Dictionary for string values specifically. Methods implemented here expect values to be either strings, or other primitives that can be converted to string implicitly (numbers, booleans, etc.).

combineWith(remoteDict) → {sntls.Dictionary}

Combines current dictionary with another dictionary and returns the combined dictionary in a new instance. The result will contain values from key-value pairs in the remote dictionary where keys match the current dictionary's values.
Parameters:
Name Type Description
remoteDict sntls.Dictionary Remote dictionary (doesn't have to be string dictionary)
Source:
  • data/StringDictionary.js, line 38
Returns:
Dictionary instance with the combined items. When the two dictionaries (current and remote) are of different (sub)classes, the return value will match the class of the remote dictionary. This way, a `StringDictionary` may be combined with a regular dictionary, resulting in a regular dictionary, but not the other way around.
Type
sntls.Dictionary
Example
var left = sntls.StringDictionary.create({foo: 'bar', hello: ['world', 'all']}),
    right = sntls.StringDictionary.create({bar: 'BAR', all: 'ALL'});
left.combineWith(right).items // {foo: "BAR", hello: "ALL"}
