Class: Dictionary

sntls. Dictionary

Manages key-value pairs. In a dictionary, one item is equivalent to a key-value pair. Internally, `Dictionary` stores key-value pairs in an object hash, its keys being dictionary keys, associated with values or arrays of values.

itemCount :number

Tracks the number of distinct key-value pairs in the dictionary. Uninitialized until first queried (by `.getItemCount()`), therefore it is safer to use its getter instead. Should not be modified externally.

items :Object|Array

Object buffer that stores items. Technically writable and public for performance and transparency reasons, but should not be changed externally as may lead to inconsistent state especially in `Hash`-based subclasses.
